# Cloud Attendance Tracking System
A very simple attendance system using Google Sheets as cloud

This project lets a user type their name and click Submit, and the attendance is saved in a Google Sheet online.
No Firebase.
No backend server.
Very easy.
100% free.

# Features

Mark attendance by typing your name

Automatically saves Name, Date, Time

Uses Google Apps Script as backend

Uses Google Sheet as database

Works online using GitHub Pages
